    The Prosocial Personality Project

      0
    Contributors:
      Anonymous Contributors

    Date created: 2020-07-21 08:55 AM | Last Updated: 2022-03-14 03:32 PM

    Identifier: DOI 10.17605/OSF.IO/M2ABP

    Category: Project

    Description: The Prosocial Personality Project collected data on various aspects of personality and related behaviors. Overall, data collection for the main (base) study included six waves run between November 2019 and April 2020. Information on sample sizes and data cleaning per wave is provided in the document "Data Cleaning & Sample Overview.pdf". All variables assessed per wave, including items and instructions, are provided in the files "Items & Materials.xlsx". The data for all waves will be successively added to this folder after all planned analyses on the data collected in a specific wave were conducted. In addition to the base study, we conducted (and are still conducting) several follow-up studies. Information on the follow-up studies is provided in the respective subfolders of this OSF project. Data collection for the base study was supported by the German Research Foundation (grant number HI 1600/1-2).
